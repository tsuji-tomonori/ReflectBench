## SnowCard（日本語）
- 要求ID: RQ-PS-001-02
- 種別: 非機能要求
- 優先度: SHOULD
- 要求: 本システムは、status API の応答を3秒以内で返却できる。
- 根拠: 運用監視の操作性を維持するため。
- 受入基準: 通常時の95パーセンタイルが3秒以内である。
- 例外/エラー: S3一時遅延時は再試行で補完できる。
- 依存・関連: [[RQ-FR-003]]
